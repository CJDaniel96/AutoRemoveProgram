# Auto Remove Program
      AutoRemoveData:
          主要是自動刪除檔案或檔案夾的程式碼和自動刪除資料庫的程式碼，運行自動刪除這動作。
      AutoRemoveProgram.bat:
          運行整個程式的批次檔
      Main.py:
          程式進入點
      MainWindow.py:
          自動刪除程式的windowForm
      Messages:
          在程式中所有的提示或警告訊息
      README.md:
          程式說明
      Settings:
          刪除時間週期設定、刪除時間點設定
      String:
          所有在該程式中用到的文字敘述
      Tray:
          程式隱藏托盤
      __init__.py:
          自動執行檔
      requirements.txt:
          該程式所有用到套件的版本
      src:
          Icon:
              該程式所有用到的圖標
          Qt:
              該程式所有UI設計的qt檔
          Table:
              該程式所有的紀錄的表格
          Log:
              該程式執行中所有刪除記錄(未開始，故未有)
